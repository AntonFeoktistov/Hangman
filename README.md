# Игра «Виселица»


Простая консольная игра на Python: угадывайте загаданное слово по буквам.

## Правила игры

- Вводите по одной **русской букве** за ход.
- Если буква есть в слове — она откроется в маске слова.
- Если буквы нет — уменьшается количество оставшихся попыток.
- Доступно **7 попыток**.
- При 0 попыток — **проигрыш**.
- Если отгадали слово до исчерпания попыток — **победа**.

## Как начать игру

1. Убедитесь, что у вас установлен Python 3.x.
2. Запустите основной скрипт:

   ```bash
   python main.py
