Игра «Виселица» 
Правила игры
Вводите по одной русской букве.
Если буква есть в слове — она откроется.
Если буквы нет — уменьшается число попыток.
При 0 попыток — проигрыш.
Если отгадали слово до исчерпания попыток — победа.


Чтобы начать игру, запустите скрипт:
bash
python main.py


Структура проекта
main.py — основной скрипт игры.
words.txt — файл со списком слов для угадывания.
pictures.py — модуль с функцией отрисовки виселицы.
